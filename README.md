# water_balance_tg_bot
This is my water balance bot created on python! This bot is really cool!

Sorry, my english is not very good, therefore it can be difficult to understand meaning of commands, 
also I would be happy if you could help me improve the bot

Commands:
  /start - starts bot and turns on notification
  /language - changes language(EN/RU)
  /help - shows main commands and gives description
  /manual - shows all functional water-tracking commands
  /status - shows amount water which user needs to drink
  /set_goal - sets the goal for drinking water
  /set_reminder - sets the time after which the notification will arrive
  /drink - records the amount of water consume
